# ktt-docker-mysql

## Credentials

```
host: 127.0.0.1
port: 3306
username: user
password: dev
```

## Usage

| command               | description                                                |
| --------------------- | ---------------------------------------------------------- |
| `docker-compose up`   | boots up the docker instance and binds to `127.0.0.1:3306` |
| `docker-compose down` | terminates and cleans the docker                           |
